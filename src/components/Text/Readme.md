## Simple text

```jsx
<Text>Simple body text</Text>
```

## Titles

```jsx
<Text as="h1" type="title1">This is title 1</Text>
<Text as="h2" type="title2">This is title 2</Text>
<Text as="h3" type="title3">This is title 3</Text>
<Text as="h4" type="title4">This is title 4</Text>
```

## Captions

```jsx
<Text as="p" type="caption-headline">This is caption headline</Text>
<Text as="p" type="caption1">This is caption 1</Text>
<Text as="p" type="caption2">This is caption 2</Text>
```

## Others

```jsx
<Text as="p" type="callout">This is callout</Text>
<Text as="p" type="subhead">This is subhead</Text>
<Text as="p" type="subtitle">This is subtitle</Text>
<Text as="p" type="subtext">This is subtext</Text>
<Text as="p" type="footnote">This is footnote</Text>
```
